Ex00: Markdown Cheatsheet.

Headings

# H1
## H2
### H3
#### H4
##### H5
###### H6

Emphasis

*italic*  or  _italic_
**bold**  or  __bold__
***bold italic***
~~strikethrough~~

Lists

Unordered:
- Item
- Item
  - Nested item

Ordered:
1. First
2. Second
3. Third

Links and images

[link text](https://example.com)
![alt text](image.png)

Blockquotes

> This is a quote
>> Nested quote

Code

Inline `code`

```python
def hello():
    print("Hello")
```

Horizontal rule

---  or  ***  or  ___

Tables

| Column A | Column B |
| -------- | -------- |
| a1       | b1       |
| a2       | b2       |
ElementMarkdown syntax
Bold**text**
Italic*text*
Link[text](url)
Inline code`code`

Task lists

- [x] Done
- [ ] Not done

Escaping

Use a backslash to escape: \*literal asterisks\*
pulgamecanica @ 42
